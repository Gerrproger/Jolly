Jolly is a modern responsive infinite carousel plugin with no dependencies that supports image zooming, fully customizable through CSS including effects which use CSS3 transitions!

Download now View on github

Features

  • Modern and lightweight carousel
  • With no dependencies
  • Fully customizable appearance through CSS
  • Lovely zooming out of the box
  • Easy customizable effects & timings (CSS3 transitions)
  • Responsive with infinite looping
  • Simple API and easy initialization (supports [data-] attributes)
  • Target element remains untouched
  • Has browser support detection

Usage

Include Jolly files

Initialize plugin with parameters

Or just use [data-] attributes

More examples and API can be found on Github
